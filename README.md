# 🚀 Setup and Run the ChatGPT Friends App
1. Installation
Make sure you install: Node.js (v18 or later) or npm (comes with Node.js). Then check the installation using this command: node -v  npm -v

2. Extract the ZIP
Extract the Zip file

4. Install Dependencies
Install the required packages using npm: npm install

6. Start the Server
Run the Node.js server: node server.js

7. Open the App in Your Browser
Go to: http://localhost:3000
